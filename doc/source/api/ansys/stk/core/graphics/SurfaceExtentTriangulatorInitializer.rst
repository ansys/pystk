SurfaceExtentTriangulatorInitializer

.. py:class:: ansys.stk.core.graphics.SurfaceExtentTriangulatorInitializer

   Bases: :py:class:`~ansys.stk.core.graphics.ISurfaceExtentTriangulatorInitializer`
